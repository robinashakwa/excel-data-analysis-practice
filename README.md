# 📊 Excel Functions Practice Portfolio

#Ongoing

## 📌 Overview

This project demonstrates the use of **commonly used Excel functions** through practical examples.
The workbook covers data analysis, formatting, logical operations, and lookup techniques used in real-world business scenarios.

---

## 📂 Sheets Included

### 1️⃣ Aggregate Functions

Used for summarizing data.

**Functions used:**

* `SUM()` → Adds values
* `AVERAGE()` → Calculates mean
* `COUNT()` → Counts numeric values
* `SUMIF()` → Adds values based on 1 condition
* `SUMIFS()` → Adds values based on multiple conditions
* `COUNTIF()` → Counts based on condition
* `COUNTIFS()` → Counts based on multiple conditions

**Use Case:**

* Sales analysis
* Performance tracking
* Reporting dashboards

---

### 2️⃣ Year-Month-Day

Extracting parts of a date.

**Functions used:**

* `YEAR()` → Extracts year
* `MONTH()` → Extracts month
* `DAY()` → Extracts day
* `ISNUMBER()` → Checks if value is a valid date

**Use Case:**

* Time-based analysis
* Monthly/Yearly reports
* Date validation

---

### 3️⃣ Formatting Functions

Used for cleaning and standardizing text.

**Functions used:**

* `TRIM()` → Removes extra spaces
* `UPPER()` → Converts text to uppercase
* `LOWER()` → Converts text to lowercase
* `PROPER()` → Capitalizes first letter of each word

**Use Case:**

* Data cleaning
* Standardizing names
* Preparing datasets

---

### 4️⃣ Logical Functions

Used for decision-making.

**Functions used:**

* `IF()` → Applies condition-based logic
* `AND()` → Checks multiple conditions
* `COUNTIF()` → Counts results based on condition

**Example:**

* Pass/Fail system based on marks

**Use Case:**

* Business rules
* Decision systems
* Validation logic

---

### 5️⃣ Lookup Functions

Used to fetch data from another table.

**Functions used:**

* `VLOOKUP()` → Finds value (left to right)
* `XLOOKUP()` → Advanced lookup (any direction)
* `IF + VLOOKUP()` → Decision based on lookup
* `IF + XLOOKUP()` → Conditional lookup logic
* `IFERROR()` → Handles errors

---

## 🔥 Key Learnings

* Difference between **calculation vs lookup**
* Importance of **clean lookup tables (unique values)**
* Handling **real-world messy data**
* Difference between **VLOOKUP and XLOOKUP**
* Combining functions like **IF + LOOKUP**

## 💡 Important Concepts

* Lookup tables must have **unique values**
* VLOOKUP has limitation: cannot lookup left
* XLOOKUP is more flexible and modern
* IF function helps in decision-making
